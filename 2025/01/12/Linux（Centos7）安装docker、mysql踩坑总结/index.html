四、总结

最后有几点需要注意：

1、重新启动虚拟机后需要再次停用原来的 mysql 防止端口冲突

2、开启 dokcer 后 mysql 并没有一并被开启，需要输入 dokcer start mysql 开启

3、如果发现数据库连接不上了可以考虑关闭虚拟机的防火墙
