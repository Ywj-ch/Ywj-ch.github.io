## 二、发现的问题

### 🔴 高优先级

| 编号 | 问题 | 影响 |
|------|------|------|
| P1 | 图片目录名含中文和特殊字符 | Git 跨平台兼容性问题 |
| P2 | 图片路径冗长 | 维护困难 |
| P3 | `.gitignore` 格式错误 | 可能误提交 |
| P4 | 配置文件全英文注释 | 理解困难 |

### 🟡 中优先级

| 编号 | 问题 | 影响 |
|------|------|------|
| P5 | JS 文件无注释 | 维护困难 |

| P7 | 缺少项目文档 | 新人上手困难 |
| P8 | 字体文件名含中文 | 兼容性问题 |

## 三、优化方案

### 3.1 图片目录重构

**方案 A（推荐）：按年份分类**
```
source/images/
├── 2025/
│   ├── leetcode-1863/
│   ├── linux-docker-mysql/
│   └── ...
├── 2024/
│   └── ...
└── common/
```

**方案 B：扁平结构**
```
source/images/
├── leetcode-1863-xor/
├── leetcode-368-subset/
└── ...
```

### 3.2 文件命名规范化

| 原文件名 | 新文件名 |
|----------|----------|
| `YSHaoShenTi-2.woff2` | `haoshenti.woff2` |
| `【哲风壁纸】海边...png` | `bg-sea.png` |
| `kslink.js` | `friend-link.js` |
| `title.js` | `title-animate.js` |
| `universe.js` | `starry-bg.js` |

### 3.3 添加文档

```
docs/
├── README.md              # 项目说明
├── deploy.md              # 部署流程
├── writing-guide.md       # 写作规范
├── image-guide.md         # 图片使用规范
└── config-guide.md        # 配置说明（中文）
```

### 3.4 配置优化

- 创建中文配置指南
- 修复 `.gitignore` 格式
- 添加 JS/CSS 文件注释
